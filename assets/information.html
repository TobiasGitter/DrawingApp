Display all layers

By clicking on selected layer again.

Clear Button

Clears selected layer or all layers if no layer selected.

Save Image Button

Saves current canvas and downloads it.

Add Layer Button

Adds a layer to paint on. New layer will be on top of existing layers.

Delete Layer Button

Deletes selected layer.

Select Area Button

Allows user to cut out part of the canvas and draw it on the canvas like a stamp.

Adjust Draw Size Slider

Allows the user to change drawing size of all drawing tools except stamp tool
that has to separate sliders for width and height.

Freehand Tool

Allows the user to draw freehand by pressing the mouse. The stroke weight is
adjustable through the slider.

LineTo Tool

Allows the user to draw straight lines by drag and drop. Once the mouse is
released the line gets drawn. The stroke weight is adjustable.

SprayCan Tool

Allows the user to spray on the canvas in a circular shape. The radius is
adjustable through the slider.

MirrorDraw Tool

Allows the user to draw symmetric freehand shapes (horizontal and vertical)
with an adjustable stroke weight.

Stamp Tool

Allows the user to stamp squares, reactangles, circles and triangles on the canvas.
Width and height are adjustable separately with to additional sliders.

Editable Shape Tool

New vertices get drawn if mouse is released. It is possible to edit the positions
of the vertices before finishing the shape.

Eraser Tool

Allows the user to erase the drawings. Important: Not really erasing but drawing
white. Lower layers will also be effected by erasing. Changing positions of the layer
with OPTION + ARROW_LEFT and OPTION + ARROW_RIGHT fixes that.

DragShape Tool

Allows the user to drag circles, squares and rectangles.

TextBox Tool

Allows the user to input text in the input field below and stamp it on the canvas.

Switch Layer Position Funcitonality

Allows user to change the position of the selected layer in the layers with with arrow buttons above the layers or OPTION + ARROW_LEFT and OPTION + ARROW_RIGHT.
